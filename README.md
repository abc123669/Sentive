# Sentive
极简句子屏保｜开源项目｜电脑/手机通用

## 项目介绍
- 时间居中超大屏保
- 鼠标静止5秒自动隐藏控件，纯时间显示
- 右下角设置：本地上传背景/链接背景/字体颜色/字体切换
- 随机句子 + 关键词搜索
- 所有设置自动保存，长期当屏保使用

## 启动方式
1. 安装 Python
2. 安装依赖：
   pip install flask flask-cors
3. 启动后端：
   python app.py
4. 打开 index.html 自动全屏使用

## 功能
✅ 自动全屏
✅ 屏保自动隐藏模式
✅ 自定义背景（本地上传/网络链接）
✅ 自定义字体颜色、字体样式
✅ 随机/搜索句子
✅ 所有设置持久化保存
✅ 极简4个文件，干净开源
